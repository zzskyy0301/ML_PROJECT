
This directory contains trained networks for the Noise2Noise paper.

Please see https://github.com/NVlabs/noise2noise for more information.

Files:

network_final-gaussian-n2c.pickle
- A network trained to denoise gaussian noise.  Trained with clean targets.

network_final-gaussian-n2n.pickle
- A network trained to denoise gaussian noise.  Trained with noisy targets (ie., noise2noise)

network_final-poisson-n2c.pickle
- A network trained to denoise poisson noise.  Trained with clean targets.

network_final-poisson-n2n.pickle
- A network trained to denoise poisson noise.  Trained with noisy targets (ie., noise2noise)
